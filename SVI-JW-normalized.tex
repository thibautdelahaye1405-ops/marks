\section{Introduction}
The standard SVI-JW parameters $(v, \psi, p, c, \tilde{v})$ are defined on the implied variance slice $v(k) = w(k)/\tau$, where $k = \ln(K/F)$ is log-moneyness and $\tau$ is time to expiry. The issue with this formulation is that the shape parameters ($\psi, p, c$) are derivatives with respect to unnormalized log-moneyness $k$, meaning they decay over time to expiry and scale with the underlying volatility level.

By transitioning from standard log-moneyness $k$ to standardized log-moneyness $z = \frac{k}{\sqrt{v \tau}}$, we can derive a parameter set that is stationary across both the term structure and different volatility regimes.

\section{The Normalized Parametrization}
The normalized SVI-JW parameter set is defined as $(v, \hat{\psi}, \hat{p}, \hat{c}, \tilde{v})$:

\begin{itemize}
    \item \textbf{ATM Implied Variance ($v$):} $v = v(0)$. Remains absolute to ensure stationarity.
    \item \textbf{Minimum Implied Variance ($\tilde{v}$):} $\tilde{v} = \min_{k} v(k)$.
    \item \textbf{Normalized ATM Skew ($\hat{\psi}$):} $\hat{\psi} = \psi \sqrt{\frac{\tau}{v}}$. This isolates the relative skew, neutralizing time decay and baseline volatility.
    \item \textbf{Normalized Put Wing ($\hat{p}$):} $\hat{p} = p \sqrt{\frac{\tau}{v}}$. Dimensionless left slope.
    \item \textbf{Normalized Call Wing ($\hat{c}$):} $\hat{c} = c \sqrt{\frac{\tau}{v}}$. Dimensionless right slope.
\end{itemize}

\section{Mapping to Raw SVI}
To price options or evaluate the variance slice, we must map the normalized parameters back to the raw Gatheral SVI parameters $(a, b, \rho, m, \sigma)$ for the total variance equation:
\begin{equation}
    w(k) = a + b \left( \rho (k - m) + \sqrt{(k-m)^2 + \sigma^2} \right)
\end{equation}

\subsection{Time/Volatility Independent Variables}
\begin{align}
    \rho &= \frac{\hat{c} - \hat{p}}{\hat{c} + \hat{p}} \\
    \beta &= \frac{\hat{c} - \hat{p} - 2\hat{\psi}}{\hat{c} + \hat{p}}
\end{align}

\subsection{Scale-Dependent Variables}
\begin{align}
    b &= \frac{1}{2} (\hat{c} + \hat{p}) \sqrt{v \tau} \\
    \sigma &= \sqrt{\frac{\tau}{v}} \left[ \frac{2(v - \tilde{v})}{(\hat{c} + \hat{p}) \left( \frac{1 - \rho \beta}{\sqrt{1-\beta^2}} - \sqrt{1-\rho^2} \right)} \right]
\end{align}

\subsection{Translation Variables}
\begin{align}
    m &= \frac{\beta \sigma}{\sqrt{1-\beta^2}} \\
    a &= \tilde{v}\tau - b\sigma\sqrt{1-\rho^2}
\end{align}

\section{Python Implementation}
Below is a robust Python implementation of the algebraic mapping, including logic to handle the $0/0$ numerical limit that occurs when ATM variance $v$ approaches the minimum variance $\tilde{v}$.

\begin{lstlisting}[language=Python]
import numpy as np

def normalized_svijw_to_raw_svi(v, v_tilde, psi_hat, p_hat, c_hat, tau):
    """
    Maps normalized cross-sectionally independent SVI-JW parameters
    to raw SVI parameters (a, b, rho, m, sigma).
    """
    # 1. Calculate time/vol independent variables
    rho = (c_hat - p_hat) / (c_hat + p_hat)
    beta = (c_hat - p_hat - 2.0 * psi_hat) / (c_hat + p_hat)
    
    # Enforce no-arbitrage bounds for beta
    beta = np.clip(beta, -0.99999, 0.99999)
    
    # 2. Calculate absolute slope parameter
    b = 0.5 * (c_hat + p_hat) * np.sqrt(v * tau)
    
    # 3. Calculate sigma (handling the v -> v_tilde singularity safely)
    eps = 1e-8
    if (v - v_tilde) < eps:
        # Smile is completely flat or perfectly symmetric at ATM
        sigma = 1e-4 
    else:
        denom_term1 = (1.0 - rho * beta) / np.sqrt(1.0 - beta**2)
        denom_term2 = np.sqrt(1.0 - rho**2)
        denominator = (c_hat + p_hat) * (denom_term1 - denom_term2)
        
        if denominator < eps:
            sigma = 1e-4
        else:
            sigma = np.sqrt(tau / v) * (2.0 * (v - v_tilde) / denominator)
            
    # 4. Calculate translation parameters
    m = (beta * sigma) / np.sqrt(1.0 - beta**2)
    a = v_tilde * tau - b * sigma * np.sqrt(1.0 - rho**2)
    
    return a, b, rho, m, sigma
\end{lstlisting}
